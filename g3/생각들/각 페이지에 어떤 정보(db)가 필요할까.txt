** 각 페이지에 필요한 것들

1. 회원가입
	- 일반회원인지, 기업회원(사업주)인지 선택 > 그에 따른 회원가입 form 변경
	- 일반회원 : 이메일,비밀번호,닉네임,이름,생년월일,성별,
	- 기업회원 : 이메일,비밀번호,업체명,사업자등록번호,대표자명,회사주소

2. 로그인
	- 일반회원,기업회원 로그인 선택(라디오박스?)

3. 검색 후 또는 일반 점포 목록 리스트 게시판 
	- 필터 설정 >>원하는 업종(한식,카페,세계요리 등등)이나 태그(혼밥,기념일,데이트)가 들어있는 가게 리스트만 볼 수 있게
	- 정렬방법 설정 고를수 있게 (최신업데이트순, 평점순,가나다순, 리뷰많은순 등등..)
	- (내 주변 지도 보기? map api 이용)
	- 각 업체당 상호명,평점,간단설명,태그,업종 정도 보이게.. (이건 회의를 통해서 정하면 될듯)

4. 가게 상세정보
	- 가게 기본정보 : 점포명, 사진들, 평점, 지역, 주소(+지도), 전화번호, 업종, 영업시간(+break시간), 대표메뉴, 상세정보, 홈페이지 주소, 태그, 조회수, 좋아요수, 스크랩수
	- 리뷰기능 : 회원들이 한줄리뷰같이 쓸 수 있는 공간 , 평점과 내용을 같이 입력
	- 좋아요, 스크랩 기능
	- 더 있으면 좋을만 한것 (관련식당 추천, 커뮤니티게시판에 관련게시글)

5. 커뮤니티 게시판
	- 리스트에 보일 것은 작성자,작성일,제목 정도..?
	- 게시글로 들어갔을때 제목,작성자,내용,사진 등등 기본정보 + 언급장소 링크, 댓글기능,좋아요기능
	- 게시글작성란에 대해서도 뭐가 들어가야할지 생각해보기  > 제목,내용,사진첨부,첨부장소 정도..?

6. 공지사항
	- 리스트에서는 제목,날짜
	- 게시글 내에서는 제목,내용,날짜, 목록으로 돌아가기
